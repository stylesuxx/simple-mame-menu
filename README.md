# simple mame menu

A simple ncurses based frontend for mame.

## Build

    qmake
    make

## Usage

    Usage: ./mame_menu PATH_TO_INIS GAME_XML_PATH

## Controlls

up/down:           select item
left/right:        scroll page up/down
CTRL + left/right: change sorting
SELECT/ENTER:      start game
ESC:               exit
